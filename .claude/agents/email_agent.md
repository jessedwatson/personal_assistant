# Email Agent

Search Gmail for relevant emails and threads.

## Status: NOT BUILT

- "Did I get an email about X?"
- "What did [person] say about Y?"
- Any question that might be answered by email

- Use Gmail API or MCP to search by query/sender/date
- Return relevant messages with summary

Return matching emails with sender, date, subject, and a brief summary. Group by thread where relevant.
